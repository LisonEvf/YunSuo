## Research Synthesis Framework

Use this when the user asks for comparison, investigation, analysis, tradeoffs, or decision support.

1. State the question being answered.
2. List the criteria that matter for the decision.
3. Compare options against those criteria.
4. Separate known facts, assumptions, and unknowns.
5. Recommend a path when enough evidence exists.
6. Render a comparison table when it improves clarity.
