## 风险控制检查清单

调用 `get_dashboard` 和 `get_market_emotion` 获取实时数据后，逐项排查。

### 一、系统性风险

- [ ] **跌停家数 > 30**：市场进入恐慌阶段，所有做多策略暂停
- [ ] **跌停连续 2 日增加**：亏钱效应扩散中，不加仓
- [ ] **两市成交额骤降 > 20%**：流动性枯竭，追高必死
- [ ] **主要指数跌破关键支撑**：系统性风险大于结构性机会

### 二、情绪风险

- [ ] **炸板率 > 40%**：追高资金严重亏损，打板策略暂停
- [ ] **封板率 < 60%**：资金分歧大，接力成功率极低
- [ ] **情绪综合指数 > 85 且连续 2 日高位**：高潮兑现风险
- [ ] **三线高度背离**（分歧度 > 20）：市场方向不明

### 三、板块风险

- [ ] **主线板块加速后缩量**：龙头见顶信号
- [ ] **板块涨停集中在补涨方向**：主线退潮，补涨尾声
- [ ] **连板高度 ≥ 7 且无板块支撑**：妖股独舞，不参与
- [ ] **板块内跌停出现**：板块内部分化加剧

### 四、个股风险

调用 `get_stock_zhangting_gene` 检查具体个股：
- [ ] **涨停基因评分 < 50**：封板能力弱，炸板概率高
- [ ] **连板高度 ≥ 4 且缩量**：接力资金不足
- [ ] **龙虎榜净卖出 > 5000 万**：大资金出逃

### 五、操作纪律

- 单票仓位不超过总仓位的 30%
- 亏损超过 5% 强制止损
- 盘中不临时加仓（除非触发计划中的加仓条件）
- 14:30 后不开新仓（尾盘不确定性高）

### 输出格式
1. 风险等级：低 / 中 / 高 / 极高
2. 触发的检查项列表（每项标注等级）
3. 建议操作：继续 / 收缩 / 离场
4. 需要重点监控的指标
